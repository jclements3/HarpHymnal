shapes/ — encoding-system docs

Rendered HTML of the shape-encoding markdown docs. The caret marks the degree (mode root) in every shape token, drawn as a LaTeX-style hat above the digit — for example 1xI333, 3RV, +2xI55. Continuation digits after the $ operator stay bare. The markdown source keeps the literal ^N form for ASCII keyboard input and grep-friendliness; the HTML build at build_html.py rewrites ^N as <span class="deg">N</span> where the inner text is the digit followed by Unicode combining circumflex U+0302. The hat is rendered by the vendored JuliaMono webfont, which anchors combining diacritics correctly above digits where most system monospace fonts (and DejaVu Sans Mono) do not. Copy-paste yields literal I, round-trippable through any Unicode-aware tool.

Quick orientation

A shape is octave? degree intervals? with an optional repeat operator at the end. The grammar production is:

shape:     octave? degree intervals? repeat?
  octave:    [+-]?[1-7][xLR]
  degree:    '^' [1-7] subscript?
  intervals: interval (,? interval){0,6}
  interval:  [2-f] subscript?
  subscript: [₁-₈]
  repeat:    (back_to_back | octave_climb) (count | * | ~)?
